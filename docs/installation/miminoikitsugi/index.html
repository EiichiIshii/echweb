耳のいきつぎ「この音はなんの音？」

インタラクティヴ・インスタレーション / Interactive installation

やまぐちアートコミュニケータープログラムの一環として開催された、フィールドレコーディングワークショップ「耳のいきつぎ」で採集された音をクイズ形式で体験するインスタレーション。
やまぐちアートコミュニケーターの藤井直子さんとの共同制作。

PC、モニター、micro:bit、ネックスピーカー、木材、プッシュスイッチほか
