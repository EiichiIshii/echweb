これは、階段に設置されたLEDテープとスピーカーから構成された作品で、光の移動と音響をbug :「虫」あるいはコンピュータにおける「バグ」両方をコンセプトとした作品である。光が移動する速度やタイミングはマイコンボードによりランダムに設定され、音響はコンセプトを元に作曲されたものである。それぞれが独立しているが、展示環境である階段で合成される形になっている。
